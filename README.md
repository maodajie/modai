#### 安装方法
1. 上传压缩包到addons文件夹
2. 解压压缩包到当前文件夹(将文件夹名称改为modai)
3. 打开苹果CMS后台->选择应用->应用市场->魔袋工具箱
4. 点击配置后自动生成快捷菜单即可使用

#### 快捷菜单
 - 魔袋工具箱,/addons/modai/index.php

#### 常见问题
 - 未生成快捷菜单请手动添加

 - 魔袋工具箱,/addons/modai/index.php
 - 魔袋工具箱,/addons/modai-mian/index.php
